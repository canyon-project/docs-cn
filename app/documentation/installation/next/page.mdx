# Next.js

import {Steps} from "nextra/components";

安装和配置 Next.js 。

<Steps>

  ### 创建项目
  首先使用以下命令创建一个新的 React 项目 `next.js` 。

  ```sh npm2yarn copy
  npm create create-next-app@latest
  ```



  ### 添加 canyon 及其配置

  > [!NOTE]
  >
  > 最新next.js编译器默认使用swc，有关babel插件的配置，请查看[这里](https://nextjs.org/docs/advanced-features/customizing-babel-config)。


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 swc 插件

  ``` js copy
  /** @type {import('next').NextConfig} */
  const nextConfig = {
  experimental: {
  swcPlugins: [
  [
  'swc-plugin-coverage-instrument', {},
  ],
  [
  // TODO: 目前SWC不支持插件内读取运行时环境变量，需要显式传入。已向SWC团队提交Issue，待解决后可删除下面的配置 https://github.com/swc-project/swc/issues/9668
  'swc-plugin-canyon', {
  projectID: process.env.CI_PROJECT_ID,
  sha: process.env.CI_COMMIT_SHA,
  dsn: process.env.DSN,
  reporter: process.env.REPORTER,
  branch: process.env.CI_COMMIT_BRANCH,
  instrumentCwd: process.cwd(),
  // compareTarget: 'dev', //可选
},
  ]
  ],
},
};

  export default nextConfig;

  ```
